# What You SHOULD Build Instead

Build this:

## “AI Audit Trail Middleware for RAG Systems”

Core idea:
Whenever AI answers a question:

* store query
* store retrieved chunks
* store final answer
* store timestamps
* generate SHA256 hash
* show logs in dashboard

That alone already sounds professional.

---

# Your MVP Features

## Phase 1 (Must Have)

### 1. Upload PDFs

* company docs
* policies
* research files

### 2. RAG Chat

* ask questions from docs

### 3. Retrieval Logging

Store:

```json
{
  "query": "",
  "retrieved_chunks": [],
  "response": "",
  "timestamp": "",
  "model": ""
}
```

### 4. SHA256 Verification

Generate:

```python
hash = sha256(query + response + chunks)
```

### 5. Dashboard

Show:

* all queries
* retrieved sources
* hashes
* timestamps

### 6. Search Audit History

Filter:

* by user
* by document
* by date
